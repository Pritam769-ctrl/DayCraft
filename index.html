DayCraft Guide

1. Planning:
Define tasks in 'Manager'. Create 'Tasksets' for routine days. In 'Planner', add individual tasks or paste sets for the next 3 days.

2. Execution:
Swipe tasks right (Done button) when completed. The vertical bar fills as you finish tasks.

3. The Debt System:
- If you miss a task, you get 1 Debt.
- To heal, you must do that task for 3 days straight (Recovery).
- Each time you miss it again, Recovery increases by 2 days.
- "Discipline is built by repair, not perfection."

4. Freeze Day ‚ùÑÔ∏è:
Found in Settings. Turn this ON during emergencies. It pauses the Debt system so you can end the day without penalties.
